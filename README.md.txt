# AI-Powered Business Intelligence Dashboard

## Project Overview
End-to-end Business Intelligence project enhanced with a Large Language Model (LLM)
to generate executive-level insights from business data.

## Tech Stack
- Python
- SQLite
- Groq LLM (LLaMA-3)
- Power BI
- Pandas, Matplotlib

## Workflow
1. Data stored in SQLite warehouse
2. Python runs analytics queries
3. LLM generates business insights
4. Insights visualized in Power BI

## How to Run
```bash
pip install pandas matplotlib groq
python python/07_llm_insights_groq.py
